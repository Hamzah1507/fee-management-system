CSV Format Required

Your CSV file must include the following columns:

  • name - Student's full name (required)

  • course - Course name (required)
  • semester - Semester number (optional)
  • dueDate - Due date in DD-MM-YYYY format (optional)
  • totalFees - Total fee amount (required, numeric)
